Hero Design 2

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Hero
  2. Custom Group Settings:
    → Set a Custom Class: 
    special-styles-hero

    → Set Fixed Hero Height (in pixels): 500
  3. Change Slide Layout and Settings → Set a Custom Class: 
    special-styles-hero-overlay-light text-center
  4. Code: Advanced → Add to Page Head Injection

Detailed Instructions

  1. Add a webpage block: Hero
  2. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-hero
  3. Click "Set Fixed Hero Height in Pixels" and enter 500 into the field
  4. Go to Slide Layout and Settings, and in the "Set a Custom Class (Advanced)" field, enter: 
    special-styles-hero-overlay-light text-center
  5. Add a background image
  6. Edit the hero content: title, description
  7. In the top-right corner of the page, click "Advanced" options
  8. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
