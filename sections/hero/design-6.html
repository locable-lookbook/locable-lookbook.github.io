Hero Design 6 - background video

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Content block → One image
  2. Custom Group Settings:
    → Set a Custom Class: 
    special-styles-hero-6
  3. Add "Embed code"
  4. Code: Advanced → Add to Page Head Injection

Detailed Instructions

  1. Add a webpage block: Content block
  2. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-hero-6
  3. Change layout by clicking arrow in the top right and choose the one column image
  4. Go to https://www.locable.com/responsive-youtube-embed-tool, paste there Youtube URL and copy from there Embed Code
  5. Go back to the editing Content Block, click image icon, click "Add Embed", paste Embed code, click "Save"
  6. Add title if needed
  7. In the top-right corner of the page, click "Advanced" options
  8. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
