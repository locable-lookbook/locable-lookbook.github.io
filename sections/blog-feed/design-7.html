Blog Feed Design 7 (Carousel, inspired by Cards Design 1)

Live Example

Step-by-step implementation

Steps Overview

  1. Webpage block: Embeds → Content / Blog Feed
  2. Create a new Blog Feed or use an existing one
  3. Custom Group Settings → Set a Custom Class: 
    special-styles-blog-feed-7
  4. Code: Advanced → Add to Page Head Injection

Detailed Instructions

  1. Add webpage block: Embeds → Content / Blog Feed
  2. If a Blog Feed is already created, skip this step. If not, create a new Blog Feed:
    1. In the modal window that opens, click "Add New"
    2. A new window titled "Blog Feed Settings" will open
    3. Fill in the fields (Name, Title, etc.) and click the "Create Embed" button
    4. Return to the page builder with the modal window still open
    5. Click "Save" in the modal window
    6. Click "Save Changes" on the page
    7. Click the orange settings button on the right and in the modal window that opens, in the field Embed, choose Blog Feed from the dropdown list
  3. In the modal window that opens, choose Blog Feed from the dropdown list
  4. Click "Save" in the modal window
  5. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-blog-feed-7

  6. Click "Save" in the modal window
  7. In the top right corner of the page, click "Advanced" options
  8. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
