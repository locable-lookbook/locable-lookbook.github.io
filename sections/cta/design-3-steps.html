Detailed Instructions

  1. Click the "Preview" button to get the page URL for later use
  2. Return to the page builder
  3. Add a webpage block: Embeds → Call-To-Action Zone
  4. If a CTA Zone does not exist, create a new CTA Zone:
    1. In the modal window that opens, click "Add New"
    2. Create CTA Zone
      • A new window titled "New CTA Zone" will open
      • Fill in the Zone Name field
      • Fill CTA Count with the number 2
      • Click the "Create" button
      • On the "Edit CTA Zone" page, click the "Create CTA" button
    3. Create First CTA:
      • Fill in the Name field
      • Check the "Active" checkbox
      • Click "Add New URL"
      • Go to the "Preview" page and copy the URL
      • Paste the URL into the "Add New URL" field
      • Click the "Create" button
      • You will be redirected to the "Edit CTA" page
      • Add the Header, Subheader, Image, URL, and Button Text
      • Click the "Save Settings" button
    4. Create Second CTA:
      • Click on "Calls-To-Action" in the left main menu
      • Click the "Create CTA" button
      • Repeat the same steps as in 4.f to set up the second CTA
    5. Return to the page builder with the modal window still open
    6. Click "Save" in the modal window
    7. Click "Save Changes" on the page
    8. Click the orange settings button on the right. In the modal window that opens, choose CTA from the dropdown list
    9. Click "Save" in the modal window
    10. Click "Save Changes" on the page
  5. If a CTA Zone was created previously (not in the step above):
    1. Click the orange settings button on the right. In the modal window that opens, choose CTA from the dropdown list
    2. Click "Save" in the modal window
    3. Click "Save Changes" on the page
    4. Inside the block titled "CTA Zone Embed", click the link with the name of the CTA
    5. A new window titled "Edit CTA Zone" will open
    6. Edit First CTA:
      • Click "Add New URL"
      • Go to the "Preview" page and copy the URL
      • Paste the URL into the "Add New URL" field
      • Click the "Save and Activate" button
    7. Edit Second CTA the same way as above, the only difference: on the Edit CTA page click checkbox Display in Following Zones to desired Zone
    8. Return to the page builder
  6. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-cta-3
  7. Click "Save" in the modal window
  8. In the top-right corner of the page, click "Advanced" options
  9. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
