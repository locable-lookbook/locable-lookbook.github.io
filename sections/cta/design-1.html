CTA Design 1

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Embeds → Call-To-Action Zone
  2. Custom Group Settings → Set a Custom Class: 
    special-styles-cta-1
  3. Code: Advanced → Add to Page Head Injection

Detailed Instructions

  1. Click the "Preview" button to get the page URL for later use
  2. Return to the page builder
  3. Add a webpage block: Embeds → Call-To-Action Zone
  4. If a CTA Zone does not exist, create a new CTA Zone:
    1. In the modal window that opens, click "Add New"
    2. A new window titled "New CTA Zone" will open
    3. Fill in the Zone Name field
    4. Click the "Create" button
    5. On the "Edit CTA Zone" page, click the "Create CTA" button
    6. Fill in the Name field
    7. Check the "Active" checkbox
    8. Click "Add New URL"
    9. Go to the "Preview" page and copy the URL
    10. Paste the URL into the "Add New URL" field
    11. Click the "Create" button
    12. You will be redirected to the "Edit CTA" page
    13. Add the Header, Subheader, Image, URL, and Button Text
    14. Click the "Save Settings" button
    15. Return to the page builder with the modal window still open
    16. Click "Save" in the modal window
    17. Click "Save Changes" on the page
    18. Click the orange settings button on the right. In the modal window that opens, choose CTA from the dropdown list
    19. Click "Save" in the modal window
    20. Click "Save Changes" on the page
  5. If a CTA Zone was created previously (not in the step above):
    1. Click the orange settings button on the right. In the modal window that opens, choose CTA from the dropdown list
    2. Click "Save" in the modal window
    3. Click "Save Changes" on the page
    4. Inside the block titled "CTA Zone Embed", click the link with the name of the CTA
    5. A new window titled "Edit CTA Zone" will open
    6. Go to the "Edit CTA" page
    7. Click "Add New URL"
    8. Go to the "Preview" page and copy the URL
    9. Paste the URL into the "Add New URL" field
    10. Click the "Save Settings" button
    11. Return to the page builder
  6. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-cta-1
  7. Click "Save" in the modal window
  8. In the top-right corner of the page, click "Advanced" options
  9. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
