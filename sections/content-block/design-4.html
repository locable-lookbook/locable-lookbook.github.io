Step-by-step implementation

Steps Overview

  1. Add a webpage block: Content Block
  2. Custom Group Settings → Set a Custom Class: 
    special-styles-content-block-4
  3. Code: Advanced → Add to Page Head Injection

Detailed Instructions

  1. Add a webpage block: Content Block
  2. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-content-block-4
  3. Add content to the block: title, description, and image
  4. In the top-right corner of the page, click "Advanced" options
  5. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
