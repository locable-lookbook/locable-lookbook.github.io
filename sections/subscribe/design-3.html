Step-by-step implementation

Steps Overview

  1. Add a webpage block: Embeds → Contact Form
  2. Custom Group Settings → Set a Custom Class: 
    special-styles-subscribe-form-3
  3. Code: Advanced → Add to Page Head Injection
  4. Add custom subscribe CSS

Detailed Instructions

  2. Create a new Contact Form:
    1. In the modal window that opens, click "Add New"
    2. A new window titled "Form Settings" will open
    3. Fill in the Name and Header (this will be the text shown in the form)
    4. In the Form Type dropdown, select Double Opt-In Newsletter Subscribe Form
    5. Keep only the Email field — remove the First Name, Last Name, and Message fields
    6. Check the "Hide Locable Branding" option
    7. In Advanced Styles, add the following custom CSS:
