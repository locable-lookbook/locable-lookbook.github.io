  3. Add background-color to Simple Text

  4. Add a webpage block: Columns → 2 columns

  6. Go to Embed part of the same column, clicking on the top right button
  7. Write there the HTML embed-responsive div: 
    <div class="embed-responsive embed-responsive-16by9"> </div>
  8. Go to Google Maps and get the Iframe code
  9. Paste the Iframe code inside the embed-responsive div
  10. Go to another column
  11. Create a form with the desired design from the Locable Lookbook Standard Form Designs.
  12. Go to Marketing Tools → Forms
  13. Go to existing form
  14. Copy Iframe Embed Code
  15. Add embed code into Embed of another column
  16. Change height of iframe, make height a little heigher then actual height
