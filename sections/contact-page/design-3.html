Step-by-step implementation

Steps Overview

  1. Add a webpage block: Simple Text for H1 title.
  2. Add a webpage block: Columns → 2 columns for links and Embed Google Map
  3. Add background-color to Columns 
    #fafafa
  4. Add pre-built Embeds → Contact Form with a class: 
    special-styles-standard-form-4
  5. Add a webpage block: Simple Text with a custom text with the same class as Contact Form: 
    special-styles-standard-form-4

Detailed Instructions

  1. Add a webpage block: Simple Text for H1 title.
  2. Add H1 Title "Contact Us"
  3. Add a webpage block: Columns → 2 columns
  4. Add background-color to Columns 
    #fafafa
  5. Add the following HTML for the address, email, phone, and social links: 
    
<h3>Get in touch</h3>

<p>
  <i class="fa fa-location-arrow p-r-1"></i>
  111 Main Street, New York, NY
</p>

<p>
  <i class="fa fa-phone p-r-1"></i>
  <a href="tel:+15551234567">+1 (555) 123-4567</a>
</p>

<p>
  <i class="fa fa-envelope p-r-1"></i>
  <a href="mailto:support@example.com">support@example.com</a>
</p>

<p>
  <i class="fa fa-clock p-r-1"></i>
  Mon–Fri: 9:00 AM – 6:00 PM
</p>

<p>Connect with us:</p>

<div style="display: flex;">
  <p style="padding: 7px;"><a href="https://facebook.com" target="_blank"><i class="fa fa-facebook"></i></a></p>
  <p style="padding: 7px;"><a href="https://instagram.com" target="_blank"><i class="fa fa-instagram"></i></a></p>
  <p style="padding: 7px;"><a href="https://youtube.com" target="_blank"><i class="fa fa-youtube"></i></a></p>
  <p style="padding: 7px;"><a href="https://tripadvisor.com" target="_blank"><i class="fa fa-tripadvisor"></i></a></p>
  <p style="padding: 7px;"><a href="https://yelp.com" target="_blank"><i class="fa fa-yelp"></i></a></p>
  <p style="padding: 7px;"><a href="https://pinterest.com" target="_blank"><i class="fa fa-pinterest"></i></a></p>
  <p style="padding: 7px;"><a href="https://linkedin.com" target="_blank"><i class="fa fa-linkedin"></i></a></p>
</div>
  6. Go to Embed part of another column, clicking on the top right button
  7. Write there the HTML embed-responsive div: 
    <div class="embed-responsive embed-responsive-16by9"> </div>
  8. Go to Google Maps and get the Iframe code
  9. Paste the Iframe code inside the embed-responsive div
  10. Create a form with the desired design from the Locable Lookbook Standard Form Designs.
  11. Add pre-built Contact Form: Embeds → Contact Form.
  12. Add the class of Contact Form, it can be any styled Standart form, for example special-styles-standard-form-4:
    Custom Group Settings → Set a Custom Class: 
    special-styles-standard-form-4
  13. Add a webpage block: Simple Text with the same class as the Contact Form: 
    special-styles-standard-form-4
    for custom text.

Code Injection

There are no special code injections for the contact page design.

Copy-paste the code injection from the chosen Locable Lookbook Standard Form Designs when you create it.
