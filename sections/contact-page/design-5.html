Steps Overview

  1. Add a webpage block: Simple Text for H1 title with class "p-b-0".
  2. Add a webpage block: Simple Text for text.
  3. Add a webpage block: Columns → 2 columns for links and Embed Google Map, and Form

    special-styles-columns-2
  6. Add pre-built Iframe with a Contact form

Detailed Instructions

  1. Add a webpage block: Simple Text for H1 title with class "p-b-0".
  2. Add H1 Title "Contact Us"
  3. Add a webpage block: Simple Text for text.
  4. Add a webpage block: Columns → 2 columns
  5. For columns in Custom Group Settings → set a custom class: 
    special-styles-columns-2
  6. Go to first column
  7. Create a form with the desired design from the Locable Lookbook Standard Form Designs.
  8. Go to Marketing Tools → Forms
  9. Go to existing form
  10. Copy Iframe Embed Code
  11. Add embed code into Embed of another column
  12. Change height of iframe, make height a little heigher then actual height
  13. Go to another column
  14. Go to Embed part of the same column, clicking on the top right button
  15. Write there the HTML embed-responsive div: 
    <div class="embed-responsive embed-responsive-16by9"> </div>
  16. Go to Google Maps and get the Iframe code
  17. Paste the Iframe code inside the embed-responsive div
  18. Add the following HTML for the address, email, phone, and social links:
