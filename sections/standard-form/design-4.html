Standard Form Design 4

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Embeds → Contact Form
  2. Custom Group Settings → Set a Custom Class to Embedded webpage block: 
    special-styles-standard-form-4
  3. Add custom CSS to embedded Standard Form
  4. Add another block with the same special class. One of webpage block: Simple text or Content block
  5. Custom Group Settings → Set a Custom Class to second webpage block: 
    special-styles-standard-form-4
  6. Code: Advanced → Add to Page Head Injection
