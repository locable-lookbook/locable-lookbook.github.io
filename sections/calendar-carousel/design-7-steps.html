Detailed Instructions

  1. Add a webpage block: Embeds → Calendar
  2. If a Calendar does not exist, create a new Calendar:
    1. In the modal window that opens, click "Add New"
    2. A new window titled "Calendar Settings" will open
    3. Fill in the fields (Name, Title, etc.)
    4. In the Choose Default View field, select Card View
    5. Click the "Save Settings" button
    6. Return to the page builder with the modal window still open
    7. Click "Save" in the modal window
    8. Click "Save Changes" on the page
    9. Click the orange settings button on the right. In the modal window that opens, choose Calendar from the dropdown list
    10. Click "Save" in the modal window
    11. Click "Save Changes" on the page
  3. If a Calendar was created previously (not in the step above):
    1. Click the orange settings button on the right. In the modal window that opens, choose Calendar from the dropdown list
    2. Click "Save" in the modal window
    3. Click "Save Changes" on the page
    4. Inside the block titled "Calendar Event Feed", click the link with the name of the Calendar
    5. A new window titled "Calendar Settings" will open
    6. In the Choose Default View field, select Card View
    7. Click the "Save Settings" button
    8. Return to the page builder
  4. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-calendar-carousel-7
  5. Click "Save" in the modal window
  6. In the top right corner of the page, click "Advanced" options
  7. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
