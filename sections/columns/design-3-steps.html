Steps Overview

  1. Add a webpage block: Columns
  2. Set up 2 columns
  3. In Custom Group Settings → set a custom class: 
    special-styles-columns-3
  4. Go to Advanced → paste code into Page Head Injection

Detailed Instructions

  1. Add a webpage block: Columns → 2 Columns
  2. Click the 3-dot icon button on the left to configure columns
  3. Click the gear icon (Settings) on the left. In the "Set a Custom Class (Advanced)" modal window, add: 
    special-styles-columns-3
  4. Add content to each column: image, link, title, description, etc.
  5. In the top-right corner of the page, click Advanced
  6. Paste the provided code snippet into the Page Head Injection field and click Save Changes
