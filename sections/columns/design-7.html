Columns 7

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Columns
  2. Set up 4 columns
  3. In Custom Group Settings → set the custom class: 
    special-styles-columns-7
  4. Add a custom background color
  5. Go to Advanced → Page Head Injection and insert the required code snippet

Detailed Instructions

  1. Add a webpage block: Columns → 4 Columns (or any other amount of columns)
  2. Click the gear icon (Settings). In the "Set a Custom Class (Advanced)" modal, enter: 
    special-styles-columns-7
  3. Add content to each column: image, link and description.
    Note: Title will not appear anywhere, in this design title doesn't exist.
  4. At the top-right of the page, click Advanced
  5. Paste the provided code snippet into the Page Head Injection field, then click Save Changes
