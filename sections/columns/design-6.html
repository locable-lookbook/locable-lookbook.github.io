Columns 6

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Columns
  2. Set up 3 columns
  3. In Custom Group Settings → set the custom class: 
    special-styles-columns-6
  4. Add custom background color
  5. Go to Advanced → Page Head Injection and insert the required code snippet

Detailed Instructions

  1. Add a webpage block: Columns → 3 Columns
  2. Click the gear icon (Settings). In the "Set a Custom Class (Advanced)" modal, enter: 
    special-styles-columns-4
  3. Click the gear icon "Adjust element specific settings": Add background color, choose darken color.
  4. Add content to each column: image, link, title, and description
  5. To hide button, but to keep all the image as a link, add class to the block clicking settings icon "Adjust element specific settings": 
    hide-button
  6. At the top-right of the page, click Advanced
  7. Paste the provided code snippet into the Page Head Injection field, then click Save Changes
