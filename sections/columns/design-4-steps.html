Steps Overview

  1. Add a webpage block: Columns
  2. Set up 2 columns
  3. In Custom Group Settings → set the custom class: 
    special-styles-columns-4

    Optionally, add one of the following classes for layout variations: 
    special-styles-columns-2
    special-styles-columns-3
  4. Add custom background color
  5. Go to Advanced → Page Head Injection and insert the required code snippet

Detailed Instructions

  1. Add a webpage block: Columns → 2 Columns
  2. Click the 3-dot icon on the left to configure the 2 columns layout
  3. Click the gear icon (Settings). In the "Set a Custom Class (Advanced)" modal, enter: 
    special-styles-columns-4
  4. To set up narrow and wide columns, add additional class: 
    special-styles-columns-2
  5. To set up wide and narrow columns, add additional class: 
    special-styles-columns-3
  6. Click the gear icon "Adjust element specific settings": Add background color, choose darken color.
  7. Add content to each column: image, link, title, and description
  8. To hide button, but to keep all the image as a link, add class to the block clicking settings icon "Adjust element specific settings": 
    hide-button
  9. At the top-right of the page, click Advanced
  10. Paste the provided code snippet into the Page Head Injection field, then click Save Changes
