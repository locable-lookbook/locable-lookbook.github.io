Columns 5 - fullscreen columns

Live Example

Step-by-step implementation

Steps Overview

  1. Add a webpage block: Columns
  2. Set up any amount of columns
  3. Custom Group Settings → Set a Custom Class: 
    special-styles-columns-5
  4. Code: Advanced → Add to Page Head Injection

Detailed Instructions

  1. Add a webpage block: Columns
  2. Set up any amount of columns
  3. Click the settings icon button on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-columns-5
  4. Add content to each column: image, link, title, and description
  5. In the top-right corner of the page, click "Advanced" options
  6. Copy the code snippet below and paste it into the "Page Head Injection" field, then click "Save Changes"
