Detailed Instructions

  1. Add a webpage block: Columns → 2 Columns
  2. Click the three-dot icon on the left to configure the layout.
  3. Click the gear (settings) icon on the left. In the "Set a Custom Class (Advanced)" modal window, add the following class: 
    special-styles-columns-2
  4. Add content to each column: image, title, description, and link (as needed).
  5. In the top-right corner of the page, click Advanced.
  6. Paste the code snippet below into the Page Head Injection field, then click Save Changes.
